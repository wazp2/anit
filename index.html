Firebase Bağlantısı

Adım adım Firebase'i kurun, ardından bilgilerinizi girin.

1. Firebase Projesi Oluşturun

  1. console.firebase.google.com adresine gidin
  2. "Add Project" → proje adı → Continue → Proje oluştur

2. Realtime Database Açın

  1. Sol menüden Realtime Database → "Create database"
  2. Konum: europe-west1 seçin
  3. "Start in test mode" seçip "Enable" tıklayın

3. Web App Config Bilgilerini Alın

  1. ⚙️ Project Settings → "Your apps" → Web (</>) simgesi
  2. Uygulama adı girin → "Register app"
  3. firebaseConfig değerlerini aşağıya yapıştırın

4. Güvenlik Kurallarını Ayarlayın

Realtime Database → Rules sekmesine tıklayın ve aşağıdaki kuralları yapıştırın:

{
  "rules": {
    "ege_hanli": {
      "approved": {
        ".read": true,
        ".write": false
      },
      "pending": {
        ".read": false,
        "songs":    { ".write": true },
        "photos":   { ".write": true },
        "messages": { ".write": true },
        "places":   { ".write": true },
        "timeline": { ".write": true }
      }
    }
  }
}

ᛝ Ziyaretçiler sadece pending/ altına yazar.
ᛝ Onaylanan içeriği Firebase Console'dan pending/ → approved/ altına kopyalayın.
ᛝ approved/ a kimse doğrudan yazamaz — sadece siz.

Demo modda veriler yalnızca bu oturumda görünür.
